# 리프트바운드 한글화 용어집 (반드시 준수)

## 규칙
- 규칙 텍스트(효과)만 번역한다. 아이콘 토큰(`:rb_might:`, `:rb_exhaust:`, `:rb_rune_rainbow:`, `:rb_rune_fury:` 등)은 **원문 그대로 유지**한다.
- 대괄호 키워드는 아래 표의 한글 키워드로 치환하되 대괄호 유지. 예: `[Legion]` → `[군단]`, `[Shield 2]` → `[보호막 2]`
- 괄호 안 리마인더 텍스트도 번역한다.
- 존댓말 없이 TCG 표준 명령문체(~한다/~합니다 대신 "~하십시오" 금지, "카드 한 장을 뽑는다" 형태).
- "me/I" = 이 카드 자신 → "나" 로 번역 (예: "When you play me" → "내가 플레이될 때").
